# Refactorings — Composing methods

Techniques for breaking up long methods, clarifying intent, and replacing opaque control flow with named steps: extract/inline method, introduce query methods, replace temp with query, split temporary variables, substitute algorithm when behavior is preserved but structure improves.

**When to use:** long methods with nested conditionals, duplicated fragments, or temps that obscure data flow.
