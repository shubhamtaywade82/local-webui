# Options Buying Knowledge Base

This repository contains a structured knowledge base for systematic options buying in the Indian markets.

## Focus Areas

- **Smart Money Concepts**: Market structure, liquidity, and institucional footprints.
- **Market Structure**: HTF/LTF alignment and BOS/CHoCH.
- **Options Chain Analysis**: OI buildup, Max Pain, and Gamma levels.
- **Volatility Models**: VIX impact and IV Rank/Percentile.
- **Options Buying Strategies**: Scalping, Momentum, and Expiry-day setups.
- **Risk Management**: Position sizing and drawdown control.
- **Algorithmic Execution**: Order execution engines and signal generation.
- **Pine Script Automation**: Custom TradingView indicators and alerts.
- **Rails API Trading Systems**: Full-stack trading automation.

## Target Audience

- **Manual Traders**: Discretionary trading with a systematic edge.
- **Algorithmic Traders**: Building robust bot architectures.
- **Backtesting Systems**: Validating models with historical data.
- **AI Trading Assistants**: RAG-ready knowledge for LLM integration.
